## 使用
   1.克隆这个repos     `$ git clone https://github.com/Fi2zz/gh-zhihu-daily.git`

   2.安装依赖          `npm install`

   3.启动服务器        `npm run server`

   4.启动开发服务器    `npm run dev`

   5.编译              `npm run build`
